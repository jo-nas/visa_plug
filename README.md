# Visa Plug for Openhtf
---


## Discription
This Package contains some basic Visa Device functionality.

## Usage
```
from visa_plug import VisaPlug
```

## Installation
```
pip install git+<repository_link>
```
or
```
git clone <repository_link>
python setup.py install
```

## Requirements
pyvisa
pyvisa-py
pyusb
pyserial
